# Evaluations

Evaluations are the core activity in the SkillForge. Managers rate team members on each skill, and employees can submit self-evaluations for comparison.

## Types of Evaluations

There are two evaluation types:

- **Manager evaluation** -- The manager assigns scores to a team member for each skill in the selected assessment period.
- **Self-evaluation** -- The team member rates their own skills. This is used for comparison against the manager's evaluation in gap analysis.

## Evaluation Flow

### 1. Select a Team Member

From the dashboard or team list, click on a team member to open their evaluation view.

### 2. Choose an Assessment

Use the assessment selector to pick the assessment period you want to evaluate. Only one assessment can be active at a time.

### 3. Enter Scores

Navigate through skill group tabs and enter or update scores for each skill. Scores are entered using a numeric input or slider, depending on the UI configuration.

::: info
Scores are not saved automatically. They are held as **pending changes** until you explicitly save them.
:::

### 4. Review Pending Changes

As you modify scores, a visual indicator shows that you have unsaved changes. You can see:

- The **current score** (previously saved value)
- The **new score** (your pending change)

This makes it easy to review what you are about to commit before saving.

### 5. Batch Save

Once you are satisfied with your changes, click the **Save** button to persist all pending changes in a single batch operation. This ensures consistency -- either all changes are saved, or none are.

::: warning
Navigating away from the evaluation page without saving will discard your pending changes. Always save before leaving.
:::

## Self-Evaluations

When a manager enables self-evaluations for an assessment, team members will see a prompt to rate their own skills. The self-evaluation uses the same skill groups and scoring scale as the manager evaluation.

Self-evaluation scores appear alongside manager scores in the radar chart and gap analysis, enabling meaningful comparison and discussion during review meetings.

## Viewing Past Evaluations

Switch the assessment selector to view evaluations from previous periods. Historical evaluations are read-only and provide a record of how skills have developed over time.
