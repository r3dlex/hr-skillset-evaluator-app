# Radar Chart

The radar chart (also known as a spider chart) provides a visual overview of a team member's skill profile across all skill groups. It is one of the most powerful tools for quickly understanding strengths and development areas.

## How to Access

The radar chart is available on the evaluation detail page for each team member. After selecting a team member and assessment, click the **Radar Chart** tab or section to view the visualization.

## Reading the Chart

The radar chart displays each skill group as an axis radiating from the center. The further a data point is from the center, the higher the score.

- **Center** = minimum score (1)
- **Outer edge** = maximum score (5)
- **Plotted area** = the team member's actual scores

Each axis is labeled with the skill group name, and the plotted polygon connects the scores to form a shape. A "rounder" shape indicates more balanced skills, while a spiky shape shows significant variation.

## Colors and Layers

The chart supports multiple overlaid datasets, each drawn in a different color:

| Layer | Color | Description |
|---|---|---|
| Manager evaluation | Primary (blue) | The manager's scores for the team member |
| Self-evaluation | Secondary (green) | The employee's self-assessed scores |
| Team average | Muted (gray) | The average score across all team members |

::: tip
Hover over a data point to see the exact numeric score for that skill group and layer.
:::

## Comparing Team Members

In some views, you can overlay multiple team members on the same radar chart. This is useful for:

- Comparing two candidates for a role
- Identifying who has the strongest profile in a particular skill group
- Spotting team-wide patterns (e.g., everyone is weak in the same area)

## Use Cases

- **Review meetings** -- Show the radar chart during one-on-ones to guide the conversation around strengths and improvement areas.
- **Team planning** -- Use the team average overlay to understand collective capability and plan training.
- **Self-awareness** -- Compare the self-evaluation layer against the manager evaluation to surface perception gaps.

::: info
The radar chart is read-only. To change scores, navigate to the evaluation view and use the score inputs.
:::
