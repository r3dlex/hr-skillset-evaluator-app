# App Tour

The HR Skillset Evaluator includes a built-in interactive tour that walks you through the main areas of the application. This is the fastest way to learn where everything is.

## Starting the Tour

There are two ways to launch the tour:

1. **From the sidebar checklist** -- When you first log in, the sidebar displays a "Getting Started" checklist. Click the "Start Tour" item to begin the guided walkthrough.
2. **From Settings** -- You can re-trigger the tour at any time from your user settings or profile area.

::: tip
If you dismissed the Getting Started checklist, you can bring it back from your profile settings.
:::

## What the Tour Covers

The tour progresses through the following steps, highlighting each area of the interface:

### Step 1: Sidebar Navigation

The tour highlights the main sidebar and explains the navigation structure -- Dashboard, Skillsets, Evaluations, Assessments, and Settings.

### Step 2: Dashboard Overview

You are shown the manager dashboard with its stats cards, team member list, and assessment selector. The tour explains how to filter by role and switch between assessments.

### Step 3: Skillsets & Skill Groups

The tour navigates to a skillset page, showing how skill groups are organized into tabs and how the "All" tab provides a consolidated table view.

### Step 4: Evaluations

You are walked through the evaluation flow -- how to select a team member, enter scores, see pending changes, and use the batch save button.

### Step 5: Radar Chart

The tour opens the radar chart view and explains how to read the multi-axis visualization, what colors represent, and how to compare team members.

### Step 6: Gap Analysis

The gap analysis view is shown, with an explanation of how team averages and role averages are calculated and what the gap values mean.

### Step 7: AI Assistant

Finally, the tour introduces the AI assistant panel -- how to open it, how it reads the current screen context, and how to use it for guided self-evaluations.

## After the Tour

Once the tour is complete, the Getting Started checklist updates to reflect your progress. You can revisit any section at any time using the sidebar navigation.

::: info
The tour does not modify any data. It is safe to run multiple times and is purely a visual guide.
:::

## Re-Triggering the Tour

If you want to run the tour again:

1. Open the sidebar
2. Look for the "Getting Started" section or navigate to your profile settings
3. Click "Restart Tour"

This is useful when onboarding new team members -- have them log in and run the tour as their first step.
