# Manager Workflow

A complete guide to evaluating your team using the HR Skillset Evaluator.

## Step 1: Import or Create Skillsets

Start by importing your team's skill matrix from an Excel file, or create skillsets manually in **Settings**.

::: tip
Importing from xlsx is the fastest way to get started. It creates skillsets, skill groups, and skills automatically.
:::

## Step 2: Create an Assessment

Navigate to any skillset and click the **+** button next to the Assessment dropdown. Name it something like "2026-Q1" or "Annual Review 2026".

## Step 3: Evaluate Team Members

1. Open a skillset from the sidebar
2. Select the assessment you created
3. Choose a **Team Member** from the dropdown
4. Switch to the **Evaluation Table** tab
5. Use the sliders to set scores for each skill
6. Click **Save Evaluation** when done

::: info Batch Save
Changes are not saved automatically. Adjust as many scores as you need, then save them all at once. You can also **Discard** changes if you change your mind.
:::

## Step 4: Review with Radar Charts

Switch to the **Radar Chart** tab to visualize scores. You can compare multiple team members by selecting "All members" in the member dropdown.

## Step 5: Analyze Gaps

Use the **Gap Analysis** tab to compare manager scores against self-evaluations, team averages, and role averages. Filter by region if needed.

## Step 6: Use the AI Assistant

Open the AI chat for help interpreting data, identifying skill gaps, or drafting evaluation feedback.

## Step 7: Export Results

Export evaluation data as xlsx for reporting, presentations, or archival.
