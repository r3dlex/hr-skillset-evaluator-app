# Onboarding

The app includes a built-in onboarding system to help new users get started quickly.

## Getting Started Checklist

When you first log in, you'll see a **Getting Started** checklist in the sidebar. It tracks your progress through essential setup steps:

### Manager Steps
1. **Import skill matrix** — Upload your team's xlsx data
2. **Review skillsets** — Browse the skillset categories
3. **Evaluate a team member** — Score a member on their skills
4. **View a radar chart** — See skills visualized
5. **Export data** — Download evaluations as xlsx

### User Steps
1. **View your scores** — Check your evaluation dashboard
2. **Complete a self-evaluation** — Rate yourself on each skill
3. **View your radar chart** — See your skills visualized
4. **Check gap analysis** — Compare self vs manager scores

## Interactive Tour

Click **Take a tour** in the Getting Started checklist to launch a guided tour of the interface. The tour highlights key UI elements with tooltips explaining each one.

You can also navigate the tour using:
- **Arrow keys** → Next / Previous
- **Escape** → Exit tour

## Dismissing the Checklist

Click **Dismiss** at the bottom of the checklist to hide it. The checklist also automatically hides once all steps are completed.

## Restarting the Tour

To restart the tour after dismissing the checklist:
- Open the **Help** page from the sidebar
- Click the tour link in the documentation

## Getting Help

- Click the **?** Help icon in the sidebar to access the full documentation
- Use the **AI Assistant** (chat bubble) for context-aware help on any page
