# Glossary

Key terms used throughout the HR Skillset Evaluator.

## Assessment
An evaluation cycle or time period (e.g., "2026-Q1", "Annual Review 2026"). Each set of evaluations belongs to an assessment.

## Evaluation
A set of scores given to a user for a specific skill within an assessment. Can include a **manager score** and a **self score**.

## Gap Analysis
A comparison view showing differences between self-evaluation scores, manager scores, team averages, and role averages.

## Manager Score
A score (0–5) assigned by a manager to a team member for a specific skill.

## Radar Chart
A spider/web chart that visually represents a user's skill scores across multiple dimensions.

## Self Score
A score (0–5) assigned by a user to themselves during self-evaluation.

## Skill
An individual competency that can be scored (e.g., "REST API Design", "Unit Testing"). Each skill has a priority level.

## Skill Group
A category grouping related skills (e.g., "Backend", "Frontend", "Soft Skills"). Skill groups belong to a skillset.

## Skillset
A top-level collection of skill groups and skills (e.g., "Fullstack / Backend", "Domain Skills"). Skillsets can be filtered by applicable roles.

## Priority
Skill importance level: **Critical**, **High**, **Medium**, or **Low**. Shown as colored badges in the evaluation table.

## Team Average
The mean manager score for a skill across all members of the selected team.

## Role Average
The mean manager score for a skill across all users with the same job title.
